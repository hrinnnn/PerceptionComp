# Videos

This directory is the intended local location for benchmark videos.

Videos are not bundled directly in this Git repository. They are currently hosted on Hugging Face at the user-provided dataset link:

- <https://huggingface.co/datasets/hrinnnn/PerceptionComp/tree/main>

Annotation entries reference videos by `video_id`, and the expected local mapping is:

```text
benchmark/videos/<video_id>.mp4
```
